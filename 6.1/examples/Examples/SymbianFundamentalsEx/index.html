Symbian OS fundamentals example code

These examples demonstrate the use of fundamental Symbian OS classes and show basic techniques that can be employed in developing Symbian OS applications.

The examples in this category are simple standalone EXEs. They use a console to display lines of text showing the behaviour of the classes.

  • HelloWorld: writes 'Hello World!' to the console
  • CClasses: typical 'C' class
  • TAndRClasses: typical 'T' and 'R' type classes
  • MClasses1-3: abstract interface classes
  • CreateStaticDLL and UseStaticDLL: using a statically linked DLL
