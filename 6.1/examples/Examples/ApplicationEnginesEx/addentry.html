AddEntry: add an Agenda entry

Found in: epoc32ex\AppEngine\AgendaModel\AddEntry

Description

This example opens an Agenda document and adds a new entry. It shows how the Agenda model can be used to edit an Agenda document via the Agenda server.

Usage

Note that the example requires the agendata3.dat data file to be located in \Epoc32\Wins\C, and it must not be read only.

Class summary

  • CAgnModel : the instance Agenda model

  • RAgendaServ : client interface to the Agenda server

  • CActiveScheduler : the active scheduler

  • RFs : file server session
