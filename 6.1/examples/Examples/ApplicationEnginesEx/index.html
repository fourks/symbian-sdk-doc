Application Engines example code

The Agenda examples show how to access an Agenda document through the Agenda model. The Agenda model can operate either in client-server or normal (direct access) mode. Client-server mode allows concurrent access to an agenda file by more than one client, and this is the mode used in these examples.

  • AddEntry: add an Agenda entry
  • ReadAppt: read Agenda appointments
  • ReadAllEntries: read all Agenda entries
  • UpdateEntry: update an Agenda entry
