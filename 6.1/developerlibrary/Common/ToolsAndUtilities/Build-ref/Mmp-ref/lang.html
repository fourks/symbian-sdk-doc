lang

lang language-list

Use the lang statement to indicate languages code for the project.

If this statement is not specified, the default language code is sc.

The language codes should be two-digit codes, and are used to complete the extension of generated resource files, as project.rsc.

Each resource file specified with resource or systemresource will be compiled multiple times, once for each language specified, though the specification of several languages will not cause several versions of the bitmaps or application information files in your project to be created.
