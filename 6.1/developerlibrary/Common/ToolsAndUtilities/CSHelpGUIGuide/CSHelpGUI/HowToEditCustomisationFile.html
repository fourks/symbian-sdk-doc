How to edit the customisation file

Before you edit the customisation file, you must specify the location of your text editor's executable file in Preferences so that the CS Help Compiler can open it for editing. See How to set project preferences.

You can edit your customisation file by selecting Advanced | Edit customisation file from the menu. The customisation file is opened in your text editor.
