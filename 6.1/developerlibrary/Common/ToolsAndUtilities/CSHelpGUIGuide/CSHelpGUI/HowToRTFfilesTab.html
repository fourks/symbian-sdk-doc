How to use the RTF files tab

Use this tab to:

  • Create a new .rtf file in a project

    Use the New button to bring up a window to choose a filename and location for your new .rtf file. The new file is then included in the list of files.

  • Add an existing .rtf file to a project

    Use the Add button to bring up a file selector dialog and add the chosen .rtf file to the project. You should have already installed the .rtf file in a sensible location, typically the project directory.

  • Edit an existing .rtf file in a project

    Use the Edit button to load the selected file into Word. Note that you must start Word first to allow your .rtf file to be loaded.

  • Remove an .rtf file from a project

    Use the Remove button to remove the selected file from the list

  • Preview a compiled .rtf file

    Use the Preview File button to compile a project containing the selected file only and preview it as XML in a web browser.
