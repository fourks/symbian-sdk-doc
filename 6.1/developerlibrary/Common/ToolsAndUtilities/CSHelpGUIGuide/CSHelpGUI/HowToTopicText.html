How to configure and create topic text

Your .rtf files use the styles in the Word template to create structured XML documents, which are then compiled into a Help file. The compiler requires certain styles to be present in the .rtf files, giving rise to a certain degree of structure.

For information on configuring a .rtf source file, see How to configure a .rtf file.
