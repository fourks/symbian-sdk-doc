How to use the Project tab

Use this tab to enter general project information. The details on this tab must be supplied before the CS Help compiler can generate any files.

Project UID

Use this to specify the unique identifier (UID) for the project. UIDs are available from the Symbian web site http://www.symbiandevnet.com.

Output folder

Use this to specify where the generated help file will go. Use the Browse button to bring up a path browser to choose this location.

Customisation file

Use this to specify the customisation file to be used. Use the Browse button to bring up a file selector dialog to choose a new customisation file. A template customisation file is included in the SDK in the location \Epoc32\cshlpcmp_template\doc\custxml.doc. Copy and paste this into your project folder.
