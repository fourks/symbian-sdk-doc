About Menu Builder

The Menu Builder is a tool designed to assist developers in the creation of menu structures for Symbian applications. The Menu builder permits the customisation of a previously created MEAD application's menu as well as the creation and edit of menus for non-MEAD generated programs.

The following functionality is provided by the Menu Builder:

  • Depicts an application menu as it will appear, in terms of general layout on an emulator/target device. Supports direct 'click and change' interaction

  • Loads the relevant files of a MEAD generated application, displays the menu as it stands and allows further modifications

  • Provides a basic menu as defaultâ€”in this case a New Menu menu with New Item menu-item

  • Allows you to insert and delete captions in top level menus

  • Allows you to insert and delete, on an existing top-level menu:

    • Standard Eikon application menu-items such as Open, Close, Copy, Paste

    • Custom, user-defined, menu-items

    • Menu separators between menu-items

  • Provides user defined menu-items that are dimmed by default, and comprise radio button group/check box variants.
