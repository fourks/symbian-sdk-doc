Overview of the Menu Builder user interface

The first step in using the Menu Builder is to create a new menu structure, or open an existing one, from the File menu. Once this has been done, use the user interface to edit and build the menu structure. The main features of the user interface are as follows:

Menu

The menu bar commands are summarised in the following table:

Menu Bar

Menu Item

Description

File

New

Creates a new menu structure. For this and other File menu commands, see How to create, load and save projects.

Open...

Opens a menubuilder.mbld menu builder file

Close

Closes the currently selected file

Save...

Saves the project file

Exit

Exits the application

Edit

Edit Current...

Executes the edit dialog for the currently selected menu or menu item

Delete current

Deletes the currently selected menu or menu item

Insert

Before Current...

Inserts a new menu before the currently selected menu

After current

Inserts a new menu after the currently selected menu

Above Current...

Inserts a new menu item above the currently selected menu item

Below Current...

Inserts a new menu item below the currently selected menu item

Separator Below Current...

Inserts a separator below the currently selected menu item

Set Insertion type

Sets a menu insertion type of one of the following: Standard Checkbox, Cascade Menu, Radio Button Group.

Tool bar

The toolbar provides quick access to selected commands:

  • New Menu Structure

  • Open Menu Definition File

  • Save Menu Structure

  • Insert Before Current/Insert Above Current

  • Insert After Current/Insert Below Current

  • Menu Item Insertion Type list box

  • Insert separator below current

  • Edit current

  • Delete current
