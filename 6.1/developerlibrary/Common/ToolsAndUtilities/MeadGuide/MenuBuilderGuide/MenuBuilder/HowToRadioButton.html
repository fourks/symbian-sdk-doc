How to insert/edit/delete a radio button group

To insert a radio button group in your menu:

  • Select a menu item

  • Set the menu insertion type using Insert | Set Insertion Type | Radio Button Group or use the listbox on the toolbar

  • Use the Insert | Above Current..., Insert | Below Current... menu options or the toolbar buttons, to insert a radio button group before or after the current selection

  • In the “Radio Button Group Editor” dialog type captions for each of the two new radio items

  • Associate each new radio item with a command and hotkey

  • Check the Select property for the radio item to be selected by default at run time. In Symbian application menus, radio buttons exist within groups containing two or more items, only one of which can be selected at run time—the editor allows only one item to be checked.

  • Set the properties:

    Enable to enable the menu item

    Append Separator to append a separator below this menu item.

  • Press OK to create the checkbox menu item

To edit an application's radio button group, use Edit | Current or the toolbar button to activate the “Radio Button Group Editor” dialog.

To delete an application's radio button group, select Edit | Delete Current... or the delete toolbar button. Note that it is not possible to delete radio buttons from a group so that less than two are present.
