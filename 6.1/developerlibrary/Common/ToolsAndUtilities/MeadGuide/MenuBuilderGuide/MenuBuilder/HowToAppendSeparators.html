How to append separators

To append a menu item separator, select the menu item below which you wish to append the separator and use Insert | Separator Below Current, Ctrl+Shift+S or the toolbar button. These methods work as a toggle to append and remove separators.

Alternatively, an Append Separator property can be set for individual menu items when they are being edited.
