How to create, load and save menu structures

To create a new standard menu structure for a non-MEAD generated application:

  • Select file | New or the toolbar button.

To save a new menu structure:

  • Select file | Save or the toolbar button.

  • In the “Save Menu” dialog, specify the Output File Paths for your Menu Definiton File, Source and Resource Files and Header File.

    These should be set to locations within your existing application's directory structure, where the compiler looks for #include files. See, How to use Menu Builder files

To open an existing menubuilder.mlbd file generated by MEAD:

  • Select file | Open... or the toolbar button.

  • Navigate to the uidef folder in your project directory created by MEAD.

  • Select the menubuilder.mbld file.

You can save edits to your menu structure using Save, and load it again later using Open....
