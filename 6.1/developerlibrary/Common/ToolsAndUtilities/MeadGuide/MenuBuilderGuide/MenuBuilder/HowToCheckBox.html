How to insert/edit/delete a checkbox menu

To insert a checkbox menu:

  • Select a menu item

  • Set the menu insertion type using Insert | Set Insertion Type | Checkbox or the listbox on the toolbar

  • Use the Insert | Above Current..., Insert | Below Current... menu options or the toolbar buttons, to insert a new checkbox menu item before or after the current selection

  • In the “Checkbox Item Editor” dialog type a caption for your checkbox menu item

  • Associate your caption with a command and hotkey

  • Select the Check property for the checkbox to be selected by default at run time.

  • Set the properties:

    Enable to enable the menu item

    Append Separator to append a separator below this menu item.

  • Press OK to create the checkbox menu item

To edit an application's checkbox menu item, use Edit | Current or the toolbar button to activate the “Checkbox Item Editor” dialog.

To delete an application's checkbox menu item, select Edit | Delete Current... or the delete toolbar button.
