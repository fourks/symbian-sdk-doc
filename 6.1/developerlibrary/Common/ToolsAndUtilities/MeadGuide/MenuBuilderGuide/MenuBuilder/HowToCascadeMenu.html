How to insert/edit/delete a cascade menu item

To insert a cascade menu item:

  • Select a menu item

  • Set the menu insertion type using Insert | Set Insertion Type | Cascade Menu or the listbox on the toolbar

  • Use the Insert | Above Current..., Insert | Below Current... menu options or the toolbar buttons, to insert a cascade menu before or after the current selection

  • In the “Cascade Menu Item Editor” dialog type a caption for the cascade menu

  • Associate each new cascade menu item with a command and hotkey

  • Set the properties:

    Enable to enable the menu item

    Append Separator to append a separator below this menu item.

  • Press OK to create the new cascade menu item.

    This creates a cascade menu with a new menu item. Edit and create new menu items using the Edit and Insert menu options or the toolbar.

To edit an application's cascade menu item, use Edit | Current or the toolbar button to activate the “Cascade Menu Item Editor” dialog.

To delete an application's cascade menu item, select Edit | Delete Current... or the delete toolbar button.
