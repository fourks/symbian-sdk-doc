Font tools

Font files provide the pixel patterns used for drawing characters. EPOC has its own font store file format (gdr).

Two methods can be used to produce a gdr file:

  • Fonts in Adobe’s Bitmap Distribution Format (bdf files) can be converted using the Fnttran tool: see Bitmap distribution format files.

  • Font specifications can be coded as a gd text file: see Font definition file. This file type can be compiled into a font store file, using the Fnttran tool.

The method available in EPOC R5 of converting fsc font formats into a gd file using the Fontcomp tool is now deprecated and the Fontcomp tool is no longer supplied with SDKs.

The font tool reference material is contained in the topics:
