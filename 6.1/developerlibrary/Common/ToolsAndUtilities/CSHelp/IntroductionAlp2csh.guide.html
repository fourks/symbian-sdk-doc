The Aleppo to CS Help converter

Overview

The Aleppo to CS Help converter transforms Aleppo project and customisation files to CS Help project and customisation files. This utility is provided to simplify the transition from the superseded Help system to CS Help.

The process

The Aleppo project and customisation files are processed by the Aleppo to CS Help converter to produce a project file in XML and a customisation file in XML. The process is shown below:

Aleppo to CS Help conversion process

The new CS Help project file contains the same information as the Aleppo project file, except for the addition of a UID field, and removal of the order-file field. The CS Help customisation file contains the same information as the Aleppo customisation file.

The Aleppo order file is not required by CS Help and is discarded by the conversion tool.
