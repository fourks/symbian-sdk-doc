Command line syntax — Aleppo to CS Help converter

Overview

The Aleppo to CS Help converter tool takes an existing Aleppo .alp project file and generates a new project and customisation file suitable for CS Help.

The output file is placed in the same directory as the original Aleppo project file, and has the name name but with an additional .xml extension. Output project files are complete except the user must add the UID for the help file into the <helpfileUID> field.

Command line syntax

alp2csh project-file.alp [-lf:logfile.txt]

Arguments

project-file

Defines the location of all source files used by the project, working directories, and target directories. Users should specify the project filename with extension. The project file must have the extension XML

logfile

If specified, the name of an optional log file
