How to use the Permanent File Store Dumper

This tool displays the content of a Permanent File Store. It is invoked as shown;

pfsdump [options] storefile

where options are;

-v

verbose: dump contents

-f

generate a frame dump

-r<n>

use nth previous store revision

-c

format output for comparison (not for frame dumps)

-q

quiet: suppress logo
