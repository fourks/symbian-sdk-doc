How to use the configuration tab

Use this tab to set the configuration options for your .sis file.

Note that the global configuration editor, which can be selected from the Tools menu, sets a default configuration for all Sisar projects. You can override the default settings for a specific .sis file using the configuration tab.

Project directory

Use this to set the directory for your project, which will contain your source files and .sisar project file. If you do not explicitly specify the project directory, Sisar will look for files relative to the location of the project field.

SIS file

Use this to select the .sis file that you are configuring.

Select the option to increment the SIS build number every time a .sis file is built, if required.

Path conversion rule

Use this to indicate how source paths should be converted to target paths. Use the Replacement path editor to select between standard locations for common file types. If the dialog does not have the desired location, you can type any path into the Path field.

SIS Signature

Use this to select the public key certificate file and private key file for the .sis file. If a passphrase exits for the .sis file, you must also supply this.
