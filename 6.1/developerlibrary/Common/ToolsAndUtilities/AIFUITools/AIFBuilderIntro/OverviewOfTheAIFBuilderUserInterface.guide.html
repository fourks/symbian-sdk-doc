Overview of the AIF Builder user interface

The first step in using AIF Builder is to create a new project file, or open an existing one, from the File menu. Once this has been done, you use the user interface to set and control the project properties. The main features of the user interface are as follows:

Menu

File: project file handling, see How to create, load and save projects

Advanced: see How to archive AIF Builder project files

Action: file generation and icon handling, see How to archive AIF Builder project files

Properties: advanced set up options, for handling new UI variants, see How to extend support to new UI variants

Tool bar

Quick access to selected commands: New, Open, Save and Edit MBM.

Dialog tabs

Use the dialog tabs to set the details of your application, and of the files that AIF Builder should generate.

  • Application: the language, name and UID of their application, see How to use the Application tab

  • UI Variants: UI target and application icon, see How to use the UI variants tab

  • Captions: application caption, see How to use the Captions tab

  • Capabilities: various advanced application properties, including MIME file type support, see How to use the Capabilities tab

Buttons

Additional buttons give quick access to commands:

  • Generate: generate the files, see How to generate the files

  • Exit: exits the application
