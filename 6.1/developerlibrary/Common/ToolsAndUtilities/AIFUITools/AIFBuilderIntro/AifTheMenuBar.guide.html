Menu summary

The menu bar commands are summarised in table below.

Menu Bar

Menu Item

Description

File

New

Creates and opens a new project file. For this and other File menu commands, see How to create, load and save projects.

Open

Opens a project file from disk

Close

Closes the currently selected file

Save

Saves the project file

Save As

Save the project file with a new name

Exit

Exits the application

Advanced

Create Zip

Create a .zip file that contains the project file and .mbm files. See How to archive AIF Builder project files.

Action

Generate

Generates the files. See How to generate the files.

Create MBM

Creates a new .mbm file. See How to provide icons.

Edit MBM

Edit the .mbm file specified on the UI Variant tab. See How to provide icons.

Load MBM

Set the .mbm file specified on the UI Variant tab

Properties

UI Variants

Allows the new UI variant's properties to be set. See How to extend support to new UI variants.

Help

About

Display the About splash screen

Help

Opens the main help topic
