How to extend support to new UI variants

AIF Builder provides default support for TechView, Quartz and Crystal, but can also be configured to support other UI variants. This configuration involves telling AIF Builder the number and format of the application icons for the UI variant.

To bring up the configuration dialog, choose the UI Variants command on the Properties menu, and enter details as follows.

Supported UI Variants

Use this to set the UI Variants supported by AIF Builder. To add support for a UI variant, choose New. You must then fill in the UI Variant Details panel for the UI variant.

Name

Use this to set the UI variant name.

Bits Per Pixel

Use this to set the colour depth of the application icons.

Icon Details

Use this to set the height and width for each application icon allowed by the UI variant. You should also enter an Icon Name: this is simply used to name the corresponding icon grid on the user interface of the AIF Icon Designer.

Note that the order that the icons are specified here sets the actual order that icons are written in a .mbm file by the AIF Icon Designer.
