WAP access point table

Support

Supported from 6.0

Description

The following macros define values for the WAP access point table fields.

The description is followed by the field type, and whether the specified field may be left null or not.

WAP_CURRENT_BEARER

WAP_CURRENT_BEARER

Description

Text string indicating the name of the table from which to read the bearer information.

This can be WAP_IP_BEARER or WAP_SMS_BEARER.

Type: Text. Null: no

See also:

  • WAP_IP_BEARER
  • WAP_SMS_BEARER

WAP_START_PAGE

WAP_START_PAGE

Description

WAP start page URL.

Type: Text. Null: yes
