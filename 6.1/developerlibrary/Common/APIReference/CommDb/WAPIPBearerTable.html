WAP IP bearer table

Support

Supported from 6.0

Description

The following macros define values for the WAP IP bearer table fields.

This table has no COMMDB_NAME field.

The description is followed by the field type, and whether the specified field may be left null or not.

WAP_ACCESS_POINT_ID

WAP_ACCESS_POINT_ID

Description

ID of the the WAP Access Point record to which this bearer information refers.

Type: TUint32. Null: no

WAP_CHARGECARD

WAP_CHARGECARD

Description

Identifier of the chargecard to use if this an ISP type database.

Type: TUint32. Null: yes

WAP_GATEWAY_ADDRESS

WAP_GATEWAY_ADDRESS

Description

WAP gateway address, an IP address in this case.

Type: Text. Null: yes

WAP_IAP

WAP_IAP

Description

Identifier of a record in the IAP table to be used.

Type: TUint32. Null: yes

WAP_ISP

WAP_ISP

Description

Identifier of the ISP to use if this an ISP type database.

Type: TUint32. Null: yes

WAP_ISP_TYPE

WAP_ISP_TYPE

Description

Whether IAP/ISP is a dial in or dial out ISP setting.

This is a TCommsDbWapIspType value.

Type: TUint8. Null: yes

See also:

  • TCommsDbWapIspType

WAP_LOCATION

WAP_LOCATION

Description

Identifier of the location to use if this an ISP type database.

Type: TUint32. Null: yes

WAP_SECURITY

WAP_SECURITY

Description

Attempt secure WTLS connection to the gateway.

Type: TBool. Null: yes

WAP_WSP_OPTION

WAP_WSP_OPTION

Description

Whether connection-oriented or connectionless API should be used.

This is a TCommsDbWapWspOption value.

Type: TUint8. Null: yes

See also:

  • TCommsDbWapWspOption
