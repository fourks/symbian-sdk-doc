WAP SMS bearer table

Support

Supported from 6.0

Description

The following macros define values for the WAP SMS bearer table fields. The description is followed by the field type, and whether the specified field may be left null or not.

This table has no COMMDB_NAME field.

The following fields in the WAP SMS bearer table are also present in the WAP IP bearer table:

  • WAP_ACCESS_POINT_ID: ID of the the WAP Access Point record to which this bearer information refers.

  • WAP_GATEWAY_ADDRESS: WAP gateway address, a phone number in this case

  • WAP_SECURITY: Attempt secure WTLS connection to the gateway

  • WAP_WSP_OPTION: TCommsDbWapWspOption value indicating whether connection-oriented or connectionless API should be used

The following field is present only in the WAP SMS bearer table:

WAP_SERVICE_CENTRE_ADDRESS

WAP_SERVICE_CENTRE_ADDRESS

Description

Service Centre Address.

Type: Text. Null: yes
