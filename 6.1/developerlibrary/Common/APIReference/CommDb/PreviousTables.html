Version 6.0 and Release 5.0 tables

Version 6.0 table

Description

The following table was present in version 6.0, but later removed.

DIAL_OUT_IAP

DIAL_OUT_IAP

Support

Supported from 6.0
Withdrawn in 6.1

Description

Dial-out Internet Access Point.

This has been replaced by the IAP table.

DIAL_OUT_IAP table fields

Description

The DIAL_OUT_IAP table contained the field IAP_CHARGECARD and the following:

IAP_ISP

IAP_ISP

Support

Withdrawn in 6.1

Description

Identifier of (dial out) ISP in this IAP

Type: TUint32. Null: no.

IAP_MODEM

IAP_MODEM

Support

Withdrawn in 6.1

Description

Identifier of modem in this IAP.

Type: TUint32. Null: yes.

IAP_LOCATION

IAP_LOCATION

Support

Withdrawn in 6.1

Description

Identifier of location in this IAP.

Type: TUint32. Null: yes.

EPOC R5 level tables

Description

The following tables were present in EPOC R5, but later removed.

MODEM_PREFS

MODEM_PREFS

Support

Withdrawn in 6.0

Description

Modem preferences.

Fields for this table are in the MODEM table in later versions.

MODEM_CONFIG

MODEM_CONFIG

Support

Withdrawn in 6.0

Description

Modem configuration.

Fields for this table are in the MODEM table in later versions.

DIAL_OUT_SERVICE

DIAL_OUT_SERVICE

Support

Withdrawn in 6.0

Description

Dial out ISP.

Replaced by DIAL_OUT_ISP.

DIAL_IN_SERVICE

DIAL_IN_SERVICE

Support

Withdrawn in 6.0

Description

Dial-in ISP.

EPOC Release 5 did not support dial-in functionality, and this is no longer documented.
