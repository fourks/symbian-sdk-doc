Chargecard table

Support

Supported from 5.0

Description

The following macros define values for the Chargecard table fields. The description is followed by the field type, and whether the specified field may be left null or not.

CHARGECARD_ACCOUNT_NUMBER

CHARGECARD_ACCOUNT_NUMBER

Description

Account number

Type: Text. Null: yes

CHARGECARD_PIN

CHARGECARD_PIN

Description

PIN number

Type: Text. Null: yes

CHARGECARD_LOCAL_RULE

CHARGECARD_LOCAL_RULE

Description

Order of dialling account number, PIN and phone number for local calls

Type: Text. Null: yes

CHARGECARD_NAT_RULE

CHARGECARD_NAT_RULE

Description

Order of dialling account number, PIN and phone number for national calls

Type: Text. Null: yes

CHARGECARD_INTL_RULE

CHARGECARD_INTL_RULE

Description

Order of dialling account number, PIN and phone number for international calls

Type: Text. Null: yes
