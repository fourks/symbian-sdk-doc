Enum TWldSelect

TWldSelect

Support

Supported from 5.0

Description

City search mode flags. These define whether a search for a city explores the whole of the world database, or just the information for a particular country. For example, the RWorldServer::NextCity() function returns either the next city in the country, or the next city in the world, depending on which enumeration is set in the TWldID.

EWithinWorld

The search is for cities within the whole world database.

EWithinCountry

The search is for cities within the current country.
