Enum TPageChange

TPageChange

Support

Supported from 5.0

Description

Flags that define how a page should be treated when it is unlocked.

EPageNoChange

Unlock only.
EPageDirty

Mark the page as dirty.
EPageUpdate

Mark the page as needing a safe update.
EPageAbandon

Discard the page.
