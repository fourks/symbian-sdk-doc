Class TAgnUniqueId

TAgnUniqueId

Support

Supported from 5.0

Description

Uniquely identifies an agenda entry or to-do list in an agenda file using an unsigned integer. This value is assigned when the entry or to-do list is added to the model, and is preserved during updates. Because of this, it can be used to identify the entry or to-do list during synchronisation.
