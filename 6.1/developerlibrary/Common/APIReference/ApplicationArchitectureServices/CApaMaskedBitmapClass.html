Class CApaMaskedBitmap

CApaMaskedBitmap

Support

Supported from 5.0

Description

An application icon.

This is a bitmap and a mask, managed by the Font and Bitmap server.

An object of this type is passed to RApaLsSession::GetAppIcon() to fetch a suitable application icon.
