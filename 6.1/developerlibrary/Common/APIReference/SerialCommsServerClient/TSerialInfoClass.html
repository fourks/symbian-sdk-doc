Class TSerialInfo

TSerialInfo

Support

Supported from 5.0

Description

Describes a serial protocol's general capabilities.

Note:

  • The lowest port need not necessarily be numbered zero.

  • At least one port is always guaranteed.

  • The full name of the serial port (used when opening it) consists of the name returned by TSerialInfo, followed by a double colon separator, followed by the unit number as an ASCII number (for instance COMM::0).
