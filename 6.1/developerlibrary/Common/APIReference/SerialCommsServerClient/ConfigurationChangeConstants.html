Configuration changed constants

Support

Supported from 6.0

Description

The following constants are used in bitmask values for TCommNotificationV01::iChangedMembers.

KRateChanged

const TUint KRateChanged

Description

The baud rate used by the remote end has changed.

KDataFormatChanged

const TUint KDataFormatChanged

Description

The data format used by the remote end has changed. At least one of the following TCommNotificationV01 members is now different: iDataBits, iStopBits, iParity.

KHandshakeChanged

const TUint KHandshakeChanged

Description

The remote handshaking has changed.
