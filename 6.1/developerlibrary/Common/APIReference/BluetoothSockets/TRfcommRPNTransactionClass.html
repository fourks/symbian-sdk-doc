Class TRfcommRPNTransaction

TRfcommRPNTransaction

Support

Supported from 6.1

Description

Utility class to specify values for one or more parameters for a Bluetooth RFCOMM port.

The class is used with TRfcommRemotePortParams::UpdateFromRPNTransaction().
