Struct THCILocalVersionIoctl

THCILocalVersionIoctl

Support

Supported from 6.1

Description

Encapsulates version information for the local Bluetooth device.

It is used with the HCI ioctl KHCILocalVersionIoctl.

For more information on the meaning of the parameters, see Read_Local_Version_Information in BS1.

Defined in THCILocalVersionIoctl:
iHCIRevision, iHCIVersion, iLMPSubversion, iLMPVersion, iManufacturerName
