Enum TRPNParameterMask

TRPNParameterMask

Support

Supported from 6.1

Description

Bitmask values used to specify parameters for a Bluetooth RFCOMM port.

See also:

  • TRfcommRPNTransaction::iParamMask
  • TRfcommRemotePortParams
EPMBitRate

Bit rate.
EPMDataBits

EPMParity

Parity type.
EPMXOnChar

XON character.
EPMXOffChar

XOFF character.
EPMXOnOffInput

XON/XOFF flow control for data input.
EPMXOnOffOutput

XON/XOFF flow control for data output.
EPMRTRInput

RTR flow control for data input.
EPMRTROutput

RTR flow control for data output.
EPMRTCInput

RTC flow control for data input.
EPMRTCOutput

RTR flow control for data output.
