Enum TRPNFlowCtrlMask

TRPNFlowCtrlMask

Support

Supported from 6.1

Description

Flags for flow control methods for a Bluetooth RFCOMM port.

The values are used with TRfcommRemotePortParams::GetFlowCtrl() and TRfcommRemotePortParams::SetFlowCtrl().

Values for input and output flow control can be combined in a bitmask.

See also:

  • TRfcommRemotePortParams::GetFlowCtrl()
  • TRfcommRemotePortParams::SetFlowCtrl()
EFCXOnOffInput

XON/XOFF flow control for data input.
EFCXOnOffOutput

XON/XOFF flow control for data output.
EFCRTRInput

RTR flow control for data input.
EFCRTROutput

RTR flow control for data output.
EFCRTCInput

RTC flow control for data input.
EFCRTCOutput

RTC flow control for data input.
