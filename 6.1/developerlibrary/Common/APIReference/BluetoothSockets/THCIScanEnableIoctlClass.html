Enum THCIScanEnableIoctl

THCIScanEnableIoctl

Support

Supported from 6.1

Description

Encapsulates the Scan_Enable parameter used to specify scanning options.

It is used with the HCI ioctl KHCIWriteScanEnableIoctl.

For more information on the meaning of the parameters, see Write_Scan_Enable in BS1.

See also:

  • KHCIWriteScanEnableIoctl
EHCINoScansEnabled

No Scans enabled.
EHCIInquiryScanOnly

Inquiry Scan enabled. Page Scan disabled.
EHCIPageScanOnly

Inquiry Scan disabled. Page Scan enabled.
EHCIInquiryAndPageScan

Inquiry Scan enabled. Page Scan enabled.
