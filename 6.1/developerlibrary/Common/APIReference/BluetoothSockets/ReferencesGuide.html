References

A large number of the items documented in this reference encapsulate commands, parameters, error codes, etc. defined in the BLUETOOTH SPECIFICATION Version 1.0 B, published at http://www.bluetooth.com. Where necessary, this is referred to in the text as [BS1].

That specification itself refers to the ETSI GSM TS 07.10 v6.3.0 standard. This is referred to in the text as [TS07].
