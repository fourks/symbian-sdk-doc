Class MIdleFindObserver

MIdleFindObserver

Support

Supported from 5.0

Description

Specifies the mixin protocol for an asynchronous find observer. An object which implements this protocol is passed to the asynchronous find functions defined in class CContactDatabase, (FindAsyncL() and FindInTextDefAsyncL()). The observer would typically notify the user of the progress of the operation.

Defined in MIdleFindObserver:
IdleFindCallback()

IdleFindCallback()

void IdleFindCallback()=0;

Description

Asynchronous find observer callback. If an observer is supplied to CContactDatabase::FindAsyncL() or FindInTextDefAsyncL(), this callback function is called by CIdle::RunL() when nothing of a higher priority can be scheduled. It is called for every 16 items searched to give the application a chance to update its search status.

The implementation of this function might typically test for and handle errors and retrieve information about the progress of the search. This information may be retrieved using functions provided by the CIdleFinder class.
