Constants

Support

Supported from 5.1

Character set identifiers

Description

These identifiers are used by the CCnvCharacterSetConverter class to identify character sets.

Notes:

  • Most of the character sets in the list below are not guaranteed to be available for conversion at run-time, and there may be character sets available at run-time that are not in this list.

  • The identifiers of all the character sets available for conversion can be found by calling CCnvCharacterSetConverter::CreateArrayOfCharacterSetsAvailableL().
