Enum TIASCharSet

TIASCharSet

Support

Supported from 5.0

Description

The character set encoding of the character string response from an IAS query.

See also:

  • TIASResponse::GetCharString()
  • TIrdaSockAddr::GetCharacterSet()

String is ISO Latin-5.
