Enum TFontPrintPosition

TFontPrintPosition

Support

Supported from 5.0

Description

Font print position flags.

Fonts can be normal, superscript or subscript.

EPrintPosNormal

Font is normal.

EPrintPosSuperscript

Font is superscript.

EPrintPosSubscript

Font is subscript.
