Resource Struct MNBUT

MNBUT

Support

Supported from 6.0

Description

A menu button.

Resources with this structure are read by CEikMenuButton::ConstructFromResourceL(), and describe the CEikMenuButton to be constructed.

See also:

  • CEikMenuButton
LLINK popout=0;

Link to a MENU_PANE resource.

See also:

  • MENU_PANE
BYTE version=0;

Version number.

WORD behavior=0;

Button behaviour flags.

See also:

  • CEikButtonBase::TButtonBehavior
WORD layout=0;

Button layout flags.

See also:

  • CEikCommandButtonBase::TLayout
LTEXT helptxt="";

Bubble help text.

See also:

  • CEikButtonBase::SetBubbleHelpTextL()
LLINK extension=0;

Unused.

LTEXT txt;

Text for the menu button.

LTEXT bmpfile="";

The .mbm file that holds an icon for the menu button.The default is the eikon .mbm file eikon.mbm.

WORD bmpid=0xffff;

ID of the icon bitmap in the specified .mbm file.The default is no icon.

WORD bmpmask=0xffff;

ID of the icon bitmap mask in the specified .mbm file.The default is no icon mask.
