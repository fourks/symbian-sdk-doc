Resource Struct NAMED_FONT

NAMED_FONT

Support

Supported from 6.0

Description

A font specified by name.

Resources of this type can be read by ResourceUtils::CreateNamedScreenFontL() to create a CFont.

See also:

  • CFont
  • ResourceUtils::CreateNamedScreenFontL()
LTEXT name;

Name of the font.

WORD height;

Height of the font in twips.

WORD flags=0;

Font flags.

See also:

  • EGulFontFlagMono
  • EGulFontFlagDoubleWidth
