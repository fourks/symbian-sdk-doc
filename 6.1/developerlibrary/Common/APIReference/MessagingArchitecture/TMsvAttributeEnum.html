Enum TMsvAttribute

TMsvAttribute

Support

Supported from 5.1

Description

Supplies values for bitmasks that allows the rapid setting or clearing of a number of TMsvEntry fields.

The fields that can be manipulated are:

  • PC synchronisation

  • Visibility flag

  • Read flag

  • In-preparation flag

  • Connected flag

  • New flag
