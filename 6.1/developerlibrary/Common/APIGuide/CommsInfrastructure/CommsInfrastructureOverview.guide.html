Comms Infrastructure Overview

Purpose

Provides frameworks and system services for communications and networking.

Description

Functionality is provided by a number of APIs. Use them as follows:

CommDb Overview

Provides system-wide storage for communications-related settings.

NifMan Overview

Allows an application to monitor network connections.

Sockets Client Overview

Provides a generic interface to communications protocols through communication end points known as sockets.

Socket Server Protocols Overview

Defines a framework for providing network services with socket type APIs.
