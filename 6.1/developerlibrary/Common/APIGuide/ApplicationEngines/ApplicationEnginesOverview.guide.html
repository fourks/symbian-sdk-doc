Application Engines Overview

Purpose

Provides access to the application data of the core applications.

The APIs allows third-party applications to integrate with the core applications: e.g. for a new communications application to read contacts data.

Description

Functionality is provided by a number of APIs. Use them as follows:

Agenda Entry and Instance Overview

The interface to individual entries and instances in an agenda file.

Agenda File Overview

The interface to the data stored in an agenda file.

Agenda Server Client Side Overview

Client side interface to the agenda server.

Agenda Model Utilities Overview

Utilities used by the agenda model-related APIs, e.g. for date and time conversion.

To-do List Overview

Manages lists of to-do entries.

Contacts Model Overview

Stores, manipulates and retrieves contacts data.
