How to use an Open Font System font

The preconditions for using an Open Font System font are that a rasterizer DLL for the font type has been installed, and that the font file containing the font has been installed.

If so, fonts from the Open Font System are accessed by clients in exactly the same way as native fonts. This process is discussed in the Fonts API topic linked to below.
