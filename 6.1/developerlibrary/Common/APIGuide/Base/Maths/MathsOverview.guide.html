Maths Overview

Purpose

Provides operations relating to real numbers.

Architectural relationships

Types for integer and single-precision and double-precision floating-point numbers are defined by the Basic Types API.

The Descriptors API provides functions that convert real numbers to strings.

Description

Functionality is provided by a number of APIs. Use them as follows:

Maths Services Overview

Provides advanced mathematical operations on real numbers.

Character Representation Of Real Numbers Overview

Defines the format of a real number type when it is converted into a string.
