How to use a notifier to display a single line message, synchronously

To display a single line message:

  • Construct an RNotifier and then call its Connect() member function to create a session with the Notifier server.

  • Call the InfoPrint() member function of the RNotifier, passing a descriptor containing the message text.

  • Close the session by calling the Close() member function of the RNotifier.

The following code fragments demonstrate this:

{

_LIT(KTxtMsg,"A message");
...
RNotifier notifier;
...
notifier.Connect();
...
notifier.InfoPrint(KTxtMsg);
notifier.Close();
...
}

The static function User::InfoPrint() is implemented using the notify server.
