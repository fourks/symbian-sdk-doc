Copying arrays

An array of non-modifiable pointer descriptors, a CPtrCArray type, provides a function which can copy elements from any descriptor array.

The source descriptor array must be one which satisfies the protocol defined by the mixin class MDesCArray.

New TPtrC elements are added to the CPtrCArray array to represent the source data. The implementation of the copy does not and cannot depend on the type of the source descriptor array, i.e. whether it is really a CPtrCArray type or a CDesCArray type.
