Using IrDA

This is an outline of the Symbian OS implementation of the standard IrDA protocol suite.

There are currently two APIs available for accessing IrDA services. These are known as the IrDA Sockets API and the IrDA CSY API. The IrDA Sockets API is presented first since it provides the programmer with access to basic IrDA functionality. The CSY API enables the programmer to transparently pipe serial port I/O over IrCOMM.
