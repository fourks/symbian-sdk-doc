Multi Media Server Overview

Purpose

Provides sound playing and manipulation, and image manipulation and format conversion.

Description

Functionality is provided by a number of APIs. Use them as follows:

Audio Sample Editor Overview

Gives clients advanced audio file manipulation behaviour; specifically, the ability to record, edit and playback sound samples.

Audio Sample Player Overview

Provides a simple interface for playing audio sample data to a default local sound device.

Audio Tone Player Overview

Provides a simple interface for tone generation (synthesised sounds) that is supported on all audio-capable devices.

Image Converter Overview

Provides a simple interface for common image-related operations.

Media Server Common Classes overview

Groups together several media server classes that are used in more than one API.
