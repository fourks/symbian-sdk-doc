Emulator syntax: epoc.exe

Purpose

The Emulator can be launched from the command line. Command line options can be used to specify the location that should be used for the emulated c: drive, and the names of the ini file and the fascia bitmap.

Command line syntax

The command line syntax is as follows:

epoc [-M<machine name>] [-T] | [-C<emulated C drive>] --

Note

  • The command line must be terminated with two hyphens.

Arguments

-C<emulated C drive>

Specifies the path of the directory that should be used for the emulated c: drive.

-T

Specifies that the system temporary path should be used as the emulated c: drive.

-M<machine name>

The base name of the .INI file and fascia bitmap. By default this is "EPOC".
