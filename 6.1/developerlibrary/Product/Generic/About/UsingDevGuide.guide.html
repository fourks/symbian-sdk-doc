Using the Developer's Guide

The Developer's Guide provides an introduction to the Symbian OS and new features of Version 6.1. It is designed to ease migration to v6.1 for developers familiar with v6.0, v5 or earlier, and to provide need-to-know information for developers new to Symbian OS.

The Practical Guide includes a Hello World example which shows off the fundamentals of the build environment, as well as detailed guide and reference topics describing essential build tools and target platforms.

All developers will find the discussions of programming conventions and essential idioms useful.
